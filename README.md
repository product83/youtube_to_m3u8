# Youtube to m3u8 <h1>
# Convert youtube playlists to m3u8 file 
# Instaling <h2> 
* git clone https://github.com/ulliss2015/youtube_to_m3u8 
* cd youtube_to_m3u8
* pip install  random
* pip install  bs4
* pip install  requests

#	How to use <h2>
* Run Playlists.py (python Playlists.py)
* Paste link
* Get your playlists.m3u8
